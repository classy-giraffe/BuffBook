# BuffBook: The Science of Size
*By classy-giraffe*

> An evidence-based manual on resistance training, muscle hypertrophy, and nutrition. Bridging the gap between complex exercise physiology and practical gym application.

## 📖 Overview
BuffBook cuts through bro-science to deliver an actionable, evidence-based system backed by peer-reviewed research. Designed with a strong focus on biomechanics, neuromechanical matching, and optimized training protocols (such as high-intensity, lower-volume approaches), this guide provides the physiological why alongside the practical how.

## 📑 Table of Contents

### Chapter 1: Foundation and Introduction
* Differentiating true myofibrillar growth from transient cell swelling.

### Chapter 2: The Biological Machinery
* Deep dive into mechanotransduction, mTORC1 pathways, and satellite cell activation.

### Chapter 3: True Drivers vs. Myths
* Why mechanical tension is the non-negotiable trigger for growth, decoupling it from metabolic stress and muscle damage.

### Chapter 4: The Specificity of Hypertrophy
* Exploring regional growth, neuromechanical matching, and the distinct roles of various contraction types.

### Chapter 5: Exercise Selection & Equipment
* A systematic approach to evaluating exercises based on stability, loadability, and resistance profiles. A critical comparison of free weights versus machines.

### Chapter 6: Programming the Variables
* Mastering the dose of training, including intensity (RIR/RPE), volume (4-12 sets), rep ranges (reps per set), Range of Motion (ROM), frequency, and progressive overload.

### Chapter 7: Practical Application & Nutrition
* Step-by-step mesocycle design, auto-regulation tactics, a complete Upper/Lower program example, and fundamentals for fueling muscle growth and facilitating fat loss.

*(A Glossary is included at the end for quick-reference definitions with relevant PubMed IDs).*

## 🚀 How to Read / Access

### Online Reading
You can read the entire manual directly on GitHub by navigating through the chapter files in the repository.

### Local Installation
To read this locally or to test changes in real time before sending a PR:
1. Clone the repository:
```bash
   git clone [https://github.com/classy-giraffe/BuffBook.git](https://github.com/classy-giraffe/BuffBook.git)
```
2. Navigate into the project directory:
```bash
   cd BuffBook
```
3. **To read as Markdown:** Open the files using your preferred Markdown viewer (e.g., Obsidian, VS Code), starting with this README.md or the first chapter file.

4. **To serve locally as a website (using Jekyll):**
   Ensure you have Ruby and Jekyll installed on your system. Then, install the dependencies and start the local server:
```bash
   bundle install
   bundle exec jekyll serve
```
Once running, open your web browser and navigate to http://localhost:4000 to view the site.
